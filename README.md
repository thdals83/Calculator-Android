# Calculator-Android
안드로이드 계산기 - 사칙연산, Mod, 비트연산, 파일입출력
- 사용언어:JAVA
- 구동환경 Android
- 사용 프로그램: Android Studio


## 주요기능
### 1) 연산
      - 사칙연산
      - 나머지 연산(MODULAR)
      - 비트 연산
      - 연산자 우선순위(괄호 포함)
      - Linked - list
      - 파일로 부터 연산
### 2) HISTORY
      - 기록 메뉴 제공
      - 기록 파일로 저장
      - 기록 계산식 삭제
      - 임의의 자리 삽입

### 3) 추가 기능
      - 입력받은 계산식의 파일 출력
      - stack 사용
